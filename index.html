How to host a static website on S3

You can use Amazon S3 to store and retrieve any amount of data at any time from anywhere on the web. This is what we'll be creating today!In this project, you'll use Amazon S3 (which stands for Amazon Simple Storage Service) to host a website.

Create S3 Bucket

Choose Bucket type.
Select The AWS region closest to you.
Enter Bucket Name. (An S3 bucket name is globally unique)

Create S3 Bucket

Make sure ACLs(Access Control Lists) are enabled.
Choose Bucket owner preferred

Create S3 Bucket

For Block Public Access settings for this bucket clear the check box for Block all public access.
