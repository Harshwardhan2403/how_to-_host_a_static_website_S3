How to host a static website on S3

You can use Amazon S3 to store and retrieve any amount of data at any time from anywhere on the web. This is what we'll be creating today!In this project, you'll use Amazon S3 (which stands for Amazon Simple Storage Service) to host a website.

Step 1

Create S3 Bucket

Choose Bucket type.
Select The AWS region closest to you.
Enter Bucket Name. (An S3 bucket name is globally unique)

Step 2

Create S3 Bucket

Make sure ACLs(Access Control Lists) are enabled.
Choose Bucket owner preferred

Step 3

Create S3 Bucket

For Block Public Access settings for this bucket clear the check box for Block all public access.

Step 4

Create S3 Bucket

Enable Bucket Versioning.
Leave the rest as it is.

Step 5

Create S3 Bucket

Create the bucket.

Step6

Create S3 Bucket

This is what it would look like.

Step 7

Upload Website Content

Open your Bucket.
Upload files into your bucket.
Make sure you do not upload files in zip format.

Step 8

Upload Website Content

You can either select files (eg:-index.html) or you can select an entire folder.

Step 9

Upload Website Content

Upload all the files and folders in your s3 Bucket.

Step 10

Upload Website Content

Uploading may take some time.
Once done you will get a notification.

Step 11

Make your Objects Public

Make your objects public else no one will be able to see them.
Select objects that you want to make public.
Click on Action.
Select make public using ACL.

Step 12

Make your Objects Public

Once done, you will get a notification.

Step 13

Static Website Hosting

Choose properties tab.
Scroll allllllllll the way down to the Static website hosting panel.
Choose edit

Step 14

Static Website Hosting

Configure the following settings: Static web hosting: Choose Enable.
Hosting type: Choose Host a static website
Index document: Enter index.html

Step 15

Static Website Hosting

Choose save change.
You will get a notification.
In the Static website hosting panel under bucket website endpoint, click on the URL.

Delete all Resources

Delete all Resources
